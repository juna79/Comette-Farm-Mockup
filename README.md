# Coomete Farm Website

A single-page food ordering website for Coomete Farm.

Customers can browse products, search by item, filter by category, add quantities to cart, view totals, open recipe ideas, add recipe ingredients, and send the final order via WhatsApp.

## Features

- Elegant farm shop homepage
- Full product catalogue
- Product images
- Search bar
- Category filters
- Deals of the day
- Add to cart
- Quantity controls
- Live cart total
- Recipe cards
- Recipe popups
- Add recipe ingredients to cart
- WhatsApp checkout
- Mobile-friendly layout
